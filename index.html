Розробка ігор – створи власну гру!

Поринь у світ розробки ігор, вивчаючи найпопулярніші технології

План навчання

1. Введення у веб-розробку та дизайн (1 місяць)

  • Як працюють сайти: клієнт-сервер, браузери
  • Основи UI/UX: що робить сайт зручним
  • Введення у Figma: інтерфейс, базові інструменти

2. HTML + CSS (2 місяці)

  • Основні теги HTML: структура сторінки
  • Стилізація з CSS: кольори, шрифти, розташування елементів
  • Flexbox та Grid: зручна верстка
  • Створення першого сайту

3. Веб-дизайн у Figma (1 місяць)

  • Створення прототипів та макетів
  • Робота з сітками та компонентами
  • Адаптивний дизайн

4. JavaScript (3 місяці)

  • Змінні, функції, події
  • Анімації та інтерактивність
  • DOM та керування елементами
  • Основи роботи з API

5. Python + Flask/Django (3 місяці)

  • Введення в програмування на Python
  • Серверна логіка та бази даних
  • Створення простого backend для сайту

6. Підсумковий проект (2 місяці)

  • Командна робота над повноцінним сайтом
  • Захист проекту та презентація

Що отримаєш після курсу?

  • Власну гру, яку можна завантажити в Google Play
  • Розуміння логіки ігрового процесу та основи кодування
  • Базові навички роботи в геймдев-індустрії

Розробка ігор – створи власну гру!

Поринь у світ розробки ігор, вивчаючи найпопулярніші технології

План навчання

1. Введение в разработку игр (1 месяц)

  • Основы гейм-дизайна и игровых механик
  • Как работают игры: игровой цикл, графика, физика
  • Знакомство с Pygame (для 2D) и Unity (для 3D)
  • Создание первого простого игрового прототипа

2. Python + Pygame – Разработка 2D-игр (3 месяца)

  • Создание окон, спрайтов и объектов
  • Управление персонажем, столкновения и анимация
  • Добавление звуков и эффектов
  • Разработка своей 2D-игры (например, платформер или арканоид)

3. Unity + C# – Разработка 3D-игр (3 месяца, для 12–16 лет)

  • Интерфейс Unity и работа с 3D-объектами
  • Основы C# для программирования логики игры
  • Физика, анимация, искусственный интеллект
  • Разработка своей 3D-игры (например, шутер или приключение)

4. Итоговый проект (3 месяца)

  • Разработка концепции игры
  • Создание уровней, персонажей и механик
  • Финальное тестирование и презентация проекта
